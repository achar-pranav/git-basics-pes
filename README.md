# Git Basics - Hello world
This repo will help you get started with basics of git, here we will learn about 

1. Using git in your day to day workflows
2. Connecting git with Github

## Getting Started
1. First fork this repository into your account
2. Then clone it locally using 
   ```bash
   git clone <repository-url>
   ```
3. Start with the [repo_setup](/repo_setup/README.md) exercise
